# 🚀 Black Space (Fekete Űr)

**Black Space** is a fast-paced, 2D top-down space shooter developed in **Unity**. This project was created as a collaborative team effort, marking a significant milestone in my professional game development journey with the Unity engine and C#.

---

### 🎮 Project Overview
In *Black Space*, players navigate through dangerous cosmic environments, battling waves of enemies and overcoming challenging obstacles. The game focuses on tight controls, satisfying combat mechanics, and a progressive difficulty system.

### ✨ Key Features
* **Intense Boss Fights:** Features **3 unique Boss encounters**, each with distinct attack patterns and phases that require tactical movement and precision.
* **Power-up System:** Players can collect various power-ups during gameplay to enhance their ship's firepower, speed, or survivability.
* **Collaborative Development:** Built as a **group project**, utilizing version control and collaborative workflows to coordinate tasks between developers.
* **Dynamic 2D Graphics:** Clean visuals, smooth animations, and optimized particle effects for an authentic arcade experience.

### 🧠 Technical Implementation
This project allowed me to dive deep into Unity's core systems:
* **C# Scripting:** Developed modular scripts for player movement, weapon systems, and enemy AI behavior.
* **Game State Management:** Handled complex transitions between menus, gameplay, and game-over states.
* **Object Management:** Efficient use of Prefabs and collision detection to manage projectiles and enemy spawns.
* **UI/UX Design:** Implemented an intuitive HUD to track health, scores, and active power-ups.

### 🛠️ Tech Stack
* **Engine:** Unity 2D
* **Programming Language:** C#
* **Platform:** PC / Desktop

### 📦 How to Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/vassko02/FeketeUr.git](https://github.com/vassko02/FeketeUr.git)
    ```
2.  Open the project folder in **Unity Hub**.
3.  Ensure you are using the appropriate Unity version (check the ProjectSettings if needed).
4.  Open the `Main` scene and press **Play**.
